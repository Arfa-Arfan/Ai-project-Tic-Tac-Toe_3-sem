# 🎮 Tic-Tac-Toe AI Game

A Python-based Tic-Tac-Toe game where you play against an AI opponent powered by the **Minimax algorithm**. Built with interactive console interface and smart AI moves.

## ✨ Features

- 🤖 **Smart AI Opponent** - Uses Minimax algorithm for unbeatable moves
- 🎯 **Two Players** - Human (X) vs AI (O)
- 📊 **Win Detection** - Automatically detects rows, columns, and diagonals
- 🤝 **Tie Detection** - Detects when board is full with no winner
- ⌨️ **User-Friendly Input** - Simple number selection (1-9) for moves
- ⏱️ **Text Animation** - Cool "Thinking..." effect for AI moves
- 🔄 **Real-time Board Display** - Updated after every move

## 🧠 AI Algorithm

The game implements the **Minimax algorithm** with depth limiting for optimal performance:

- **Maximizing Player**: AI (O) tries to win
- **Minimizing Player**: Human (X) represented in algorithm
- **Depth Limit**: 5 moves for faster computation
- **Evaluation**: Returns 1 (AI win), -1 (Human win), or 0 (tie)
